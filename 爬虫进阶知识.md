## 一、JavaScript 逆向工程

现代网站大量使用 JS 加密、签名、混淆来保护接口，JS 逆向是进阶爬虫最核心的技能。

### 1.1 认识加密参数的来源

打开 Chrome DevTools → Network → 找到目标 XHR 请求 → 查看 Payload：

```json
{
  "page": 1,
  "sign": "a8f3b9c2d1e4567890abcdef...",
  "timestamp": 1702345678,
  "token": "eyJhbGciOiJIUzI1NiJ9..."
}
```

`sign`、`token` 这类参数就是逆向目标——找到它们的生成逻辑并在 Python 中复现。

### 1.2 定位加密逻辑的方法

#### 方法一：XHR 断点（最常用）

```
1. DevTools → Sources → XHR/fetch Breakpoints
2. 添加断点 URL 片段（如 "api/list"）
3. 刷新页面，触发断点
4. 在 Call Stack 中从下往上追溯
5. 找到加密函数所在位置
```

#### 方法二：全局搜索

```
1. DevTools → Search (Cmd+Opt+F)
2. 搜索参数名，如 "sign"、"encrypt"、"getToken"
3. 在搜索结果中逐文件排查
```

#### 方法三：Initiator 追溯

```
1. Network → 点击目标请求
2. 查看 Initiator 标签页
3. 点击调用链中的文件链接
4. 逐层向上追溯
```

#### 方法四：Hook 关键函数


### 1.3 常见 JS 混淆与应对

| 混淆方式 | 特征 | 应对工具 |
|----------|------|----------|
| **变量名混淆** | `_0x1234`, `a`, `b`, `c` | AST 还原、手动重命名 |
| **字符串数组** | `_0xabc1('0x1', 'abcd')` | 提取字符串表、运行还原 |
| **控制流平坦化** | switch-case 无限分支 | AST 反混淆、符号执行 |
| **JSVMP/代码虚拟化** | 自定义字节码解释器 | 极难还原，考虑补环境执行 |
| **obfuscator.io** | 标准混淆器特征 | `synchrony`、`webcrack` 等工具 |
| **sojson/JSFuck** | 花指令、eval 嵌套 | 动态调试、AST 解包 |

### 1.4 补环境执行（RPC 调用）

当 JS 依赖浏览器环境（window、document、navigator 等）时，直接在 Python 中复现成本极高。可以采用"补环境"或"RPC"方式：让加密逻辑在浏览器中执行，Python 通过协议与之通信。

```python
# 方案：使用 Playwright + 注入JS 实现 RPC
from playwright.sync_api import sync_playwright
import json

def get_sign(page, data):
    """调用页面中的加密函数"""
    encrypted = page.evaluate(f'''
        () => {{
            return window.encryptData({json.dumps(data)});
        }}
    ''')
    return encrypted

with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page()
    page.goto('https://target-site.com')
    page.wait_for_load_state('networkidle')
    
    result = get_sign(page, {'page': 1, 'keyword': 'test'})
    print(result)
    
    browser.close()
```

#### 更优雅的方案：js2py / PyExecJS / Node.js 子进程

```python
import subprocess
import json

def js_sign(data):
    """通过 Node.js 执行 JS 加密"""
    js_code = '''
    const crypto = require('crypto');
    function sign(data) {
        const str = JSON.stringify(data) + 'secret_key';
        return crypto.createHash('md5').update(str).digest('hex');
    }
    console.log(JSON.stringify(sign(%s)));
    '''
    result = subprocess.run(
        ['node', '-e', js_code % json.dumps(data)],
        capture_output=True, text=True
    )
    return json.loads(result.stdout.strip())
```

### 1.5 实战案例：从混淆 JS 中提取加密逻辑

```python
"""
逆向思路：某网站 sign 参数逆向

1. 抓包发现请求携带 sign=md5(data + timestamp + salt)
2. XHR 断点定位到 app.min.js 中的 _0x4a2b 函数
3. 发现 salt 是固定值 "d#@$F2s9"
4. Python 复现：
"""

import hashlib
import time

SALT = "d#@$F2s9"

def generate_sign(params):
    raw = json.dumps(params, sort_keys=True) + str(int(time.time() * 1000)) + SALT
    return hashlib.md5(raw.encode()).hexdigest()
```

### 3.3 自建分布式爬虫

```python
import redis
import json
import hashlib
from multiprocessing import Process
import requests
from bs4 import BeautifulSoup

class DistributedCrawler:
    def __init__(self, redis_url='redis://localhost:6379/0'):
        self.redis = redis.from_url(redis_url)
        self.task_queue = 'crawler:tasks'
        self.visited_set = 'crawler:visited'
        self.result_list = 'crawler:results'
    
    def push_task(self, url, depth=0, meta=None):
        """推送任务到队列"""
        task = {'url': url, 'depth': depth, 'meta': meta or {}}
        self.redis.lpush(self.task_queue, json.dumps(task))
    
    def is_visited(self, url):
        """布隆过滤器去重"""
        url_hash = hashlib.md5(url.encode()).hexdigest()
        return self.redis.sismember(self.visited_set, url_hash)
    
    def mark_visited(self, url):
        url_hash = hashlib.md5(url.encode()).hexdigest()
        self.redis.sadd(self.visited_set, url_hash)
    
    def save_result(self, data):
        self.redis.lpush(self.result_list, json.dumps(data, ensure_ascii=False))
    
    def crawl_worker(self, worker_id):
        """爬虫工作进程"""
        print(f'Worker {worker_id} started')
        while True:
            # 阻塞式弹出任务
            _, task_raw = self.redis.brpop(self.task_queue, timeout=5)
            if not task_raw:
                continue
            
            task = json.loads(task_raw)
            url = task['url']
            
            if self.is_visited(url):
                continue
            
            try:
                resp = requests.get(url, timeout=10, headers={
                    'User-Agent': 'Mozilla/5.0 ...'
                })
                self.mark_visited(url)
                
                # 解析、提取数据...
                self.save_result({'url': url, 'status': resp.status_code})
                
                # 发现新链接
                soup = BeautifulSoup(resp.text, 'lxml')
                for a in soup.find_all('a', href=True):
                    new_url = a['href']
                    if task['depth'] < 3:
                        self.push_task(new_url, task['depth'] + 1)
                        
            except Exception as e:
                print(f'Error on {url}: {e}')
    
    def run(self, start_urls, num_workers=4):
        """启动分布式爬虫"""
        for url in start_urls:
            self.push_task(url)
        
        processes = []
        for i in range(num_workers):
            p = Process(target=self.crawl_worker, args=(i,))
            p.start()
            processes.append(p)
        
        for p in processes:
            p.join()
```

#### 图像预处理增强识别率

```python
from PIL import Image, ImageFilter, ImageEnhance
import numpy as np

def preprocess_captcha(image_bytes):
    """提高 OCR 识别率的预处理流程"""
    from io import BytesIO
    img = Image.open(BytesIO(image_bytes)).convert('L')  # 灰度
    
    # 二值化
    img = img.point(lambda x: 0 if x < 128 else 255, '1')
    
    # 去噪
    img = img.filter(ImageFilter.MedianFilter(3))
    
    # 放大（OCR 对小字识别差）
    img = img.resize((img.width * 3, img.height * 3), Image.LANCZOS)
    
    return img
```

```python
def find_gap_position(bg_path, slider_path):
    """找到滑块缺口位置"""
    bg = cv2.imread(bg_path)

    # 返回水平偏移
    return max_loc[0]

def generate_track(distance):
    """生成模拟人类滑动的轨迹"""
    track = []
```

### 5.2 无限滚动页面

```python
async def crawl_infinite_scroll(url, max_items=100):
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        page = await browser.new_page()
        await page.goto(url)
        
        items = []
        prev_count = -1
        
        while len(items) < max_items:
            # 滚动到底部
            await page.evaluate('window.scrollTo(0, document.body.scrollHeight)')
            
            # 等待新内容加载
            await asyncio.sleep(2)
            
            # 重新统计
            items = await page.query_selector_all('.list-item')
            
            if len(items) == prev_count:
                break  # 没有新内容了
            prev_count = len(items)
        
        # 提取数据
        results = []
        for item in items[:max_items]:
            title = await item.query_selector('.title')
            results.append(await title.text_content())
        
        await browser.close()
        return results
```

### 5.3 拦截 API 请求直接获取数据

```python
async def intercept_api(url):
    api_data = []
    
    async with async_playwright() as p:
        browser = await p.chromium.launch(headless=True)
        page = await browser.new_page()
        
        # 拦截所有 XHR/fetch 响应
        async def handle_response(response):

                    api_data.append(data)
                except:
                    pass
```

```python
class ProxyPool:
    """简易代理池"""

        self.redis = redis.from_url('redis://localhost:6379/1')
        self.proxy_key = 'proxy_pool'

        """从代理源获取代理（示例：免费代理API）"""
        urls = [
            'https://proxylist.free-proxy-list.net/proxylist.json',  # 示例
            # 付费代理源...
        ]
        for url in urls:
            try:
                resp = requests.get(url, timeout=10)
                proxies = resp.json()
                for p in proxies:
                    self.redis.sadd(self.proxy_key, f"{p['ip']}:{p['port']}")
            except:
                continue
    
    def check_proxy(self, proxy):
        """检测代理可用性"""
        try:
            resp = requests.get(
                self.check_url,
                proxies={'http': f'http://{proxy}', 'https': f'http://{proxy}'},
                timeout=5
            )
            if resp.status_code == 200:
                return True
        except:
            pass
        self.redis.srem(self.proxy_key, proxy)
        return False
    
    def validate_all(self):
        """批量校验代理"""
        proxies = [p.decode() for p in self.redis.smembers(self.proxy_key)]
        with ThreadPoolExecutor(max_workers=20) as executor:
            executor.map(self.check_proxy, proxies)
    
    def get_proxy(self):
        """随机获取一个可用代理"""
        proxy = self.redis.srandmember(self.proxy_key)
        return f'http://{proxy.decode()}' if proxy else None
    
    def run(self, interval=300):
        """定时更新+校验"""
        while True:
            print('Fetching proxies...')
            self.fetch_proxies()
            print('Validating proxies...')
            self.validate_all()
            print(f'Pool size: {self.redis.scard(self.proxy_key)}')
            time.sleep(interval)
```

### 6.3 智能重试与退避

```python
import time
import random

class SmartRetry:
    def __init__(self, max_retries=5, base_delay=1, max_delay=60):
        self.max_retries = max_retries
        self.base_delay = base_delay
        self.max_delay = max_delay
    
    def should_retry(self, status_code):
        """判断是否需要重试"""
        retry_statuses = {429, 500, 502, 503, 504}
        return status_code in retry_statuses
    
    def get_delay(self, attempt, status_code=None):
        """指数退避 + 随机抖动"""
        delay = min(self.base_delay * (2 ** attempt), self.max_delay)
        jitter = random.uniform(0, delay * 0.3)
        return delay + jitter
    
    def execute(self, func, *args, **kwargs):
        for attempt in range(self.max_retries):
            try:
                resp = func(*args, **kwargs)
                if resp.status_code == 200:
                    return resp
                
                if self.should_retry(resp.status_code):
                    delay = self.get_delay(attempt, resp.status_code)
                    print(f'状态码 {resp.status_code}，{delay:.1f}s 后重试 ({attempt+1}/{self.max_retries})')
                    time.sleep(delay)
                    continue
                
                return resp  # 不重试的状态码
            
            except requests.RequestException as e:
                delay = self.get_delay(attempt)
                print(f'请求异常 {e}，{delay:.1f}s 后重试 ({attempt+1}/{self.max_retries})')
                time.sleep(delay)
        
        raise Exception(f'重试 {self.max_retries} 次后仍失败')

# 使用
retry = SmartRetry()
resp = retry.execute(requests.get, 'https://example.com/api', headers=headers)
```

```python
import re

class TextCleaner:
    @staticmethod

            elif code == 0x3000:
                result += ' '

                result += char
```

### 9.2 海量数据去重 — 布隆过滤器

```python
import mmh3
import math
from bitarray import bitarray

class BloomFilter:
    """布隆过滤器 — 内存高效的 URL 去重"""
    
    def __init__(self, capacity=100_000_000, error_rate=0.001):
        """
        capacity: 预期元素数量
        error_rate: 误判率
        """
        self.size = int(-capacity * math.log(error_rate) / (math.log(2) ** 2))
        self.hash_count = int(self.size / capacity * math.log(2))
        self.bit_array = bitarray(self.size)
        self.bit_array.setall(0)
    
    def _hashes(self, item):
        """生成多个哈希值"""
        if isinstance(item, str):
            item = item.encode('utf-8')
        for i in range(self.hash_count):
            yield mmh3.hash(item, i) % self.size
    
    def add(self, item):
        for h in self._hashes(item):
            self.bit_array[h] = 1
    
    def contains(self, item):
        return all(self.bit_array[h] for h in self._hashes(item))

# 使用
```
